# Task E - Compliance and Supply Chain (Shop application)

This folder contains the Flask shop application and SQLite assets used for Task E (SBOM, containerisation, and SSDF-aligned supply-chain documentation).

## File layout (as requested)

| File | Purpose |
|---|---|
| `app.py` | Flask web application |
| `database.db` | SQLite database (generated by `init_db.py`) |
| `init_db.py` | Creates schema and seed data from `schema.sql` |
| `readme.txt` | Local run instructions |
| `schema.sql` | Database schema |
| `requirements.txt` | Python dependencies for installs and SBOM tooling |
| `static/css/style.css` | Stylesheet referenced by `app.py` |

## Quick start

From this directory:

```bash
python init_db.py
pip install -r requirements.txt
flask --app app run
```

Open `http://127.0.0.1:5000/`.

Default admin user (from `init_db.py`): username `admin`, password `password` (intentionally weak for lab use only).
